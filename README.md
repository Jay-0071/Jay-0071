- **Real-Time Emotion Detection:** Analyzes facial expressions from live webcam feeds using a streamlined CNN model trained on the FER2013 dataset.
- **Multilingual NLP & Voice Synthesis:** Features advanced NLP translation and high-quality, emotion-infused Text-To-Speech (TTS) utilizing the ElevenLabs API and gTTS.
- **Robust Full Stack Architecture:** Designed with a high-performance **FastAPI** Python backend coupled directly with a modern, responsive **React** frontend for a seamless user experience.
